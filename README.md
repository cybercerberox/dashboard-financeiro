# 📊 Dashboard Financeiro

Uma plataforma moderna de gestão financeira desenvolvida para transformar dados financeiros complexos em informações visuais simples, intuitivas e acessíveis.

O projeto simula sistemas utilizados por fintechs e plataformas de controle financeiro, oferecendo dashboards interativos, métricas em tempo real e uma experiência moderna para gerenciamento financeiro pessoal ou empresarial.

---

## 🚀 Objetivo

O principal objetivo da aplicação é facilitar o controle financeiro através de uma interface visual moderna, permitindo que usuários acompanhem receitas, despesas, indicadores e comportamento financeiro de forma prática e eficiente.

---

## ✨ Funcionalidades

- 📈 Dashboard financeiro interativo
- 💳 Controle de receitas e despesas
- 📊 Gráficos e métricas em tempo real
- 📱 Interface responsiva
- ⚡ Experiência moderna e otimizada
- 🧠 Visualização simplificada de dados financeiros

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- Python
- Streamlit
- Pandas
- Plotly
- HTML/CSS
